Deux types de câbles de couleur sont utilisés sur ce véhicule. Le premier type est d'une seule couleur et le second est bicolore (rayé).
Pour les câbles d'une seule couleur, un seul symbole de couleur est utilisé (p. ex. "GRN").
Pour les câbles bicolores, deux symboles de couleur sont utilisés (p. ex. "GRN/YEL"). Le premier symbole représente la couleur de base du câble ("GRN" sur le schéma). Le second symbole représente la couleur de la rayure ("YEL" sur le schéma).
